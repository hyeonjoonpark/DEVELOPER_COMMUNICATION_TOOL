### STACKS

| Stack          | Stack Detail                                    |
|----------------|-------------------------------------------------|
| Language       | JAVA 17                                         |
| Framework      | Spring Framework (Spring Boot, Spring Security) |
| Database       | MySQL, Redis                                    |
| Message Stream | Kafka                                           |
| Security       | Oauth2 Client, JWT                              |
| etc.           | WebSocket, WebRTC, GraphQL(Apollo Server)       |
| DevOps         | Docker, Kubernetes                              |
| CI / CD        | Github Action                                   |
